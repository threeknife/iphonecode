- (BOOL) intersectsMapRect: (BMKMapRect)  mapRect [optional]

判断指定的矩形是否与本Overlay相交，用于更精确的控制overlay view的显示. 默认使用BMKMapRectIntersectsRect([overlay boundingRect], mapRect)代替.

参数:
mapRect指定的BMKMapRect
返回:
如果相交返回YES，否则返回NO
